How I've Made This Website



This website is a project- here I've been able to explore methods of web design from planning a website to executing the site in code (no help from Wordpress or Weebly here).

Coding this site from scratch has taught me a lot about design. I learned that I couldn't just guess at how I wanted my website to look. I really needed a drawn out design. Unfortunately I couldn't afford to use the Adobe Suite to create a design, so my final mock-up was on WXYZ.com
